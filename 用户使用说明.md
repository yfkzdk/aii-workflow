# AII上下文助手 - 用户使用说明

## 这个工具是做什么的？

帮你把任务交给 Claude Code，自动生成方案、选最优解、执行并保存结果。遇到报错还能自动重试、断点恢复。

---

## 第一步：启动

打开命令行，输入：

```
cd "O:\AII\上下文助手"
python ww_fixed.py "你的任务描述"
```

**举例：**
```
python ww_fixed.py "帮我写一个读取CSV的Python脚本"
```

启动后，系统会自动生成一段指令，复制粘贴到 Claude Code 即可。

---

## 第二步：查看结果

任务完成后，结果保存在：
```
O:\AII\上下文助手\tasks\output_result.md
```

---

## 如果中途报错了（400错误）

不用慌，系统会自动重试最多3次。

如果还是不行：
1. 关掉当前 Claude Code 窗口
2. 打开一个新窗口
3. 运行：`python ww_fixed.py recover`
4. 按提示选择要恢复的任务，复制恢复指令到新窗口

---

## 其他常用命令

| 想做什么 | 输入什么 |
|---------|---------|
| 查看当前任务状态 | `python ww_fixed.py status` |
| 恢复中断的任务 | `python ww_fixed.py recover` |
| 清理7天前的旧数据 | `python ww_fixed.py clean 7` |
| 查看版本 | `python ww_fixed.py version` |

---

## 也可以用图形界面

双击 `极简启动器.bat`，按数字菜单操作即可，无需输入命令。

---

## 注意事项

- 任务描述要**简短明确**，3-5句话最好
- 大任务建议拆成小任务逐个执行
- 如果显示乱码，先运行 `问题解决工具.bat`，选择"修复编码环境"
- **不要使用** `windows启动器.bat` 和 `ww_simple.py`，它们有已知问题

---

## 核心文件一览

| 文件 | 作用 |
|------|------|
| `ww_fixed.py` | 主启动器，推荐使用 |
| `极简启动器.bat` | 图形菜单版 |
| `问题解决工具.bat` | 诊断+修复工具 |
| `tasks/output_result.md` | 任务结果 |
| `config/user_prefs.json` | 个人设置 |
| `workflows/` | 所有任务记录 |

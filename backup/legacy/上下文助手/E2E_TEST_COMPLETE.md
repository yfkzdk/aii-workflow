## 📋 使用指南

### 快速开始
```bash
# 1. 初始化系统
python scripts/workflow_manager.py init

# 2. 创建新任务
python scripts/workflow_manager.py create \
  --title "优化Python函数" \
  --goal "提高calculate_total函数的性能" \
  --constraints "保持向后兼容性，不改变函数签名" \
  --expected "性能提升20%以上"

# 3. 生成方案
python scripts/workflow_manager.py solutions --task TASK-YYYYMMDD-HHMMSS

# 4. 方案对比
python scripts/workflow_manager.py compare --task TASK-YYYYMMDD-HHMMSS

# 5. 运行测试
python workflows/tests/e2e_test.py
```

### 与Claude Code集成
1. **主调度器配置**: 将 `.claude/CLAUDE.md` 内容复制到Claude Agent配置
2. **任务工作流**:
   - 用户提出需求
   - 调度器创建 `tasks/input_task.md`
   - 工作流代理生成三个方案
   - 对比选择最优方案
   - 实施并归档结果
   - 调度器返回压缩结果 (≤300 tokens)

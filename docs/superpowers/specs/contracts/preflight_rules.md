# PreFlight 校验规则表

## 版本兼容性校验（SemVer）

| 配置版本 | 运行时版本 | 兼容性 | 匹配规则 | 说明 |
|---------|-----------|--------|---------|------|
| 1.0.0 | 1.0.0 | ✅ 兼容 | `^1.0.0` | 主版本相同，次版本向上兼容 |
| 1.0.0 | 1.2.5 | ✅ 兼容 | `^1.0.0` | 主版本相同，运行时次版本更高 |
| 1.0.0 | 2.0.0 | ❌ 不兼容 | 主版本不同 | 主版本必须相同 |
| 1.2.0 | 1.1.0 | ❌ 不兼容 | 运行时版本过低 | 运行时版本必须 >= 配置版本 |
| ~1.2.0 | 1.2.5 | ✅ 兼容 | `~1.2.0` | 主+次版本锁定，补丁版本向上兼容 |
| ~1.2.0 | 1.3.0 | ❌ 不兼容 | 次版本不同 | `~` 锁定主+次版本 |
| >=1.0.0 | 2.0.0 | ✅ 兼容 | `>=1.0.0` | 显式范围匹配 |

## 变量绑定校验规则

| 变量名 | 类型 | 是否关键 | 默认值 | 缺失行为 | 校验规则 |
|-------|------|---------|--------|---------|---------|
| plan.id | string | ✅ 关键 | 无 | 阻断启动 | 非空字符串，UUID 格式 |
| user.role | string | ✅ 关键 | 无 | 阻断启动 | 枚举值：admin/developer/viewer |
| stage.name | string | ✅ 关键 | 无 | 阻断启动 | 枚举值：input_collecting/executing/verifying/archiving |
| project.security_level | string | ❌ 非关键 | "standard" | 使用默认值 + warn 日志 | 枚举值：low/standard/high/critical |
| task.id | string | ❌ 非关键 | 自动生成 | 使用默认值 + warn 日志 | UUID 格式 |

## 显式存在性校验（区分 None 与假值）

| 输入值 | `is None` | `or 默认值` | 正确行为 | 错误行为 |
|-------|-----------|------------|---------|---------|
| `None` | ✅ True | 使用默认值 | 阻断（关键变量）或使用默认值（非关键） | - |
| `0` | ❌ False | ❌ 误判为缺失 | **保留原值 0** | ~~覆盖为默认值~~ |
| `False` | ❌ False | ❌ 误判为缺失 | **保留原值 False** | ~~覆盖为默认值~~ |
| `""` | ❌ False | ❌ 误判为缺失 | **保留原值 ""** | ~~覆盖为默认值~~ |
| `"valid"` | ❌ False | 保留原值 | 保留原值 | - |

## 配置文件校验规则

| 校验项 | 规则 | 错误码 | 处置动作 |
|-------|------|--------|---------|
| 文件存在性 | 文件必须存在且可读 | CFG-001 | 阻断启动 |
| JSON/YAML 语法 | 必须符合语法规范 | CFG-002 | 阻断启动 |
| Schema 校验 | 必须符合 JSON Schema | CFG-002 | 阻断启动 |
| 尾随空格清理 | 所有字符串键/值去除首尾空格 | - | 自动清理 |
| 非法配置组合 | `critical=true` 且 `on_fail=skip` | CFG-003 | 阻断启动 |
| 占位符白名单 | 仅允许预定义变量 | CFG-005 | 阻断启动 |
| 版本兼容性 | SemVer 范围匹配 | CFG-006 | 阻断启动 |

## 启动耗时要求

| 场景 | 耗时要求 | 输出要求 |
|-----|---------|---------|
| 合法配置通过 | < 500ms | ✅ 启动前校验通过 |
| 非法配置拦截 | < 1s | ❌ 启动前校验失败 + 结构化错误报告 |
| 错误报告格式 | JSON | `{"errors": [{"code": "CFG-001", "message": "...", "location": "..."}]}` |
